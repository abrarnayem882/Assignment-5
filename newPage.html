Question-1: What are the different ways to select an element in the DOM?

  1. 1. document.getElementById('id') - Selects by ID.
  2. 2. document.getElementsByClassName('class') - Selects by class.
  3. 3. document.getElementsByTagName('tag') - Selects by tag name.
  4. 4. document.querySelector('selector') - Selects the first matching element.
  5. 5. document.querySelectorAll('selector') - Selects all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  1. * innerHTML - Returns or sets HTML content.
  2. * innerText - Returns or sets visible text.
  3. * textContent - Returns or sets all text.

Question-3: What is event delegation in the DOM?

Answer: A technique where a parent element handles events for dynamically added child elements using event.target. This improves performance by reducing event listeners.

Question-4: What is event bubbling in the DOM?

Answer: A process where an event on a child element propagates up through its ancestors in the DOM tree, triggering their respective event handlers unless stopped using event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

  1. # Create: document.createElement('tag')
  2. # Add: parent.appendChild(child) or parent.append(child)
  3. # Remove: element.remove() or parent.removeChild(child)
